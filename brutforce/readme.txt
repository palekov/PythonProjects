
План создания проекта:

1. Создаем в Pycharm новый проект с новым именем
2. Создаем три файла: users.json - где хранятся имена пароли, и два питоновских - сервер и перебор. 
3. Наполняем все три файла кодом и данными
4. Запускаем через run сначала сервер (проверяем что он работает на 127.0.0.1:5000)
5. Запускаем брут (проверяем, что перебор идет):

3 17722 doa
3 17723 dob
3 17724 doc
3 17725 dod
3 17726 doe
3 17727 dof
3 17728 dog
Success!

и проверяем, что сервер корректно логгирует:

127.0.0.1 - - [19/May/2020 19:45:34] "POST /auth HTTP/1.1" 401 -
127.0.0.1 - - [19/May/2020 19:45:34] "POST /auth HTTP/1.1" 401 -
127.0.0.1 - - [19/May/2020 19:45:34] "POST /auth HTTP/1.1" 401 -
127.0.0.1 - - [19/May/2020 19:45:34] "POST /auth HTTP/1.1" 200 -

Так же можно зайти в браузер: http://127.0.0.1:5000/
